# Gemini Context

## Project Overview
This project is a personal static portfolio website for Shiva Chavoshian. 
The goal is to present a professional, modern, and aesthetically pleasing portfolio that showcases UI/UX design portfolio to hiring managers HR recruiters. I also have some artwork that shows my skill in art and design.

## Project Objectives
- **Professional Presentation**: The site should be visually appealing and professional.
- **Modern Design**: The site should be modern and dynamic with micro interactions.
- **Responsive Design**: The site should be responsive and mobile-friendly.
- **Content Management**: The site should be easy to update and maintain.
- **Performance**: The site should be fast and responsive.
- **Engaging Experience**: The site should be easy to understand and folllow and engaging because I want to show my skill in UI/UX design and art.

## Constraints
- **Static Site**: The site must remain static (HTML/CSS/JS).
- **No Complex Build System**: Updates are done manually checking in files. Avoid heavy bundlers or complex CI/CD pipelines for content updates.
- **Maintainability**: Code should be clean, well-organized, and easy to understand for manual editing.
- **Design**: "Modern, beautiful, and dynamic". High visual quality and engaging experience and micro interactions is a priority.

## Tech Stack Guidelines
- **HTML**: Semantic HTML5.
- **CSS**: Modern CSS (Flexbox, Grid, CSS Variables). 
- **JS**: Modern Vanilla JS (ES6+).
- **Assets**: Optimized images and fonts.
